## Run Locally
**Prerequisites:** Node.js (version 18+ recommended)

1. Install all dependencies (this uses `package.json` to pull everything automatically):
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key.
3. Start the app:
   ```bash
   npm run dev
   ```
